# 📹 Управление видео — Инструкция

## Как управлять видео на сайте

### Шаг 1 — Запустите сервер загрузки

Откройте **Терминал** в папке с сайтом и выполните:

```
npm run server
```

Оставьте окно открытым. Вы увидите:
```
🚀 Сервер загрузки запущен: http://localhost:3001
🔑 Пароль: starlink2025
```

### Шаг 2 — Откройте Админ-панель

Откройте браузер и перейдите на:
```
http://localhost:5173/admin
```
*(если сайт запущен через `npm run dev`)*

### Шаг 3 — Войдите с паролем

Пароль: **starlink2025**

> ⚠️ Пароль можно изменить: откройте файл `server.js`, найдите строку  
> `const ADMIN_PASSWORD = process.env.ADMIN_PASSWORD || 'starlink2025'`  
> и замените `starlink2025` на свой пароль.

---

## Что можно делать в Админ-панели

| Действие | Как |
|---|---|
| **Загрузить новое видео** | Нажмите кнопку «Загрузить» на нужном слоте, выберите файл |
| **Заменить видео** | Загрузите новый файл в тот же слот — старый заменится |
| **Изменить название** | Кликните на текстовое поле с названием и напишите новое |
| **Скрыть видео** | Нажмите иконку 👁 — слот скроется с сайта |
| **Удалить видео** | Нажмите иконку 🗑 — файл удалится, слот скроется |

---

## Технические детали

- Видеофайлы хранятся в папке `public/`
- Конфигурация (названия, порядок, вкл/выкл) — в файле `public/videos.json`
- Максимальный размер файла: **500 МБ**
- Поддерживаемые форматы: **MP4, MOV, WebM**

---

## Быстрая замена файла вручную (без Админ-панели)

Если вы умеете работать с файлами:
1. Переименуйте своё видео в `video1.mp4` (или `video2.mp4` ... `video6.mp4`)
2. Скопируйте его в папку `public/`
3. Старый файл с таким же именем заменится
